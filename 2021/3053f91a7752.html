  • 2021-07-18

netfox-zxkit

一个查看app中所有网络请求的库，依赖NetFox功能

该库已经集成到ZXKitSwift中，用于记录网络请求，集成ZXKitSwit，即可在调试的工具列表中有该功能

网络拦截请求效果截图

自定义安装

如果你自定义选择安装该库，可以根据后面的集成和注册进行安装使用，如果您已经集成了ZXKitSwift，只需要更新pod库就会更新上该功能

cocoapods

pod 'netfox-zxkit'

import netfox_zxkit
//注册 regist
ZXKit.regist(plugin: NetFoxZXKit())

Display the list of tools

ZXKit.show()

hide the list of tools

ZXKit.hide()

close ZXKit

ZXKit.close()
