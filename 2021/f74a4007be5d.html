iOS UserDefault数据管理-ZXUserDefaultManager

  • 2021-07-18
  • DamonHu

iOS UserDefault data management, iOS UserDefault数据管理，可以查看、编辑、删除app中的UserDefault数据

该项目是ZXKitSwift的内置功能插件，也可以单独集成当做独立功能使用。

如果您已经集成了ZXKitSwift，插件列表中会自动显示该数据管理浏览器，不需要再重复集成。

独立集成

cocoapods快速集成

pod 'ZXUserDefaultManager'

使用

//显示管理器
ZXUserDefaultManager.shared.start()

ZXKit个性化集成

如果您已经集成了ZXKitSwift，插件列表中会自动显示该文件浏览器，不需要再重复集成。

如果需要支持ZXKit，可以使用cocoapods快速集成

pod 'ZXUserDefaultManager/zxkit'

之后可在AppDelegate的启动函数中注册到ZXKit即可

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    
    //ZXKit注册
    ZXKit.regist(plugin: ZXUserDefaultManager.shared)
    
    return true
}
